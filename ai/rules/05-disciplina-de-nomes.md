# Rule 05 — Disciplina de nomes

**Faça:** usar `snake_case` para módulos Python e para pastas de modelo
(`models/<nome>/`). O nome do config bate exatamente com o nome do modelo
(`config/<nome>.*` ↔ `models/<nome>/`).
**Não faça:** `CamelCase`/`kebab-case` em módulos ou pastas de modelo; nome de config
divergente do nome do modelo.
