# Rule 07 — Pipeline config-driven

**Faça:** ler todo caminho, nome de tabela, hiperparâmetro e nome de recurso do
`config/`. O código recebe valores; não os inventa.
**Não faça:** hardcodar no código qualquer caminho, tabela, hiperparâmetro, nome de
experimento ou nome de fonte de dado. Se está fixo no `.py`, está no lugar errado —
mova para o config.
