# Rules deste projeto

As rules do **molde** (inegociáveis, verificadas pelo CI) moram em
[`docs/rules/`](../../docs/rules/) — leia-as antes de mexer em qualquer coisa.

Esta pasta é para rules **específicas deste projeto** (acréscimos que só valem aqui).
Enquanto não houver nenhuma, ela fica só com este aviso. Não duplique as rules do molde
aqui; elas continuam valendo e sendo checadas pelo CI de onde estão.
